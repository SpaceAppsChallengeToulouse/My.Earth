RESTful API

Get Data

URL : http://www.mygeopostcard.com/earthday/area
Http Method : GET

Create Data

URL : http://www.mygeopostcard.com/earthday/area
Http Method : POST
Payload : 
{
	author: "NASA",
	description: "My Form Post",
	endDate: 946684800000,
	points: [{
		coordinates:{
			lat:34.9,
			lon:-118.09
		}//nb coordinates == 1 => Marker, nb coordinates > 2 => Polygon
	}],
	startDate: 631152000000,
	type: "Air pollution",
	value: "600"
}

Delete Data

URL : http://www.mygeopostcard.com/earthday/area/:id
Http Method : DELETE
