MY.EARTH (client-site)

Context

We develop this project for the NASA international space apps challenge. More information here: http://spaceappschallenge.org/.


MY.EARTH is our solution for the Earth Day Challenge.

Description

This is the MY.EARTH client-side part.


It provides a clean web application that displays the evolution of environmental data on Earth since 1970.

Technology

This project uses awesome open source technologies:

  • AngularJS
  • Bootstrap
  • Cordova
  • jQuery
  • Leaflet
